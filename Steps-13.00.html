Adding project Logo’s and Title 13

Adding Colaberry Logo:  

1.    Insert: Image and select Colaberry logo from the downloaded file
2.    Go to General tab and set Properties: Size: Height = 90, Width = 273

                                      Position: Horizontal = 0, Vertical = 0
Adding Booking Method Logo: 

1.    Insert: Image and Select Booking Method Logo from the downloaded file.
2.    Go to General tab and set Property: Size: Height = 90, Width =273

                                      Position: Horizontal = 1003, Vertical = 0
Adding Title: 

1.    Insert a Text Box and insert “Travel Booking and Revenue.”
2.    The Font and style will be set to Segoe UI, 28
3.    Go to General tab and set Property: Size: Height = 91, Width = 729

                                      Position: Horizontal = 273 Vertical = 0
