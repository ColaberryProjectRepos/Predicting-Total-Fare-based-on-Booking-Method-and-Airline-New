Creating 5th Field Map Visual (Fare By Depart Country) 21

 ·       From the Visualizations pane, select the Filled Map.
·        In Values, drag and drop the = DepartCountry_x and DepartCity_x.
·        Tooltips = Passenger and Avg. Fare
        Tooltip = Keep all filters on.
Format Visual Tab:

·         Click on Format visual tab, Map Settings:  Style-Light Controls Auto Zoom, On; Zoom on selection, Off; Manual Zoom, Off.
·         Legend: Off. 
 Fill colors : Callout value: Font = DIN, 45, Color, Green = #00F45A, Yellow = #E4F001, Red = F70010, Display units = Auto, Category label = OFF
General: Property: Size: Height = 319, Width = 502
                                          Position: Horizontal = 776, Vertical = 400
Title: ON, Title Fare by Depart Country, Font = DIN, 14, Text Color =  Black (#252423), Background Color = (#04B9AC), Horizontal Alignment = Center
Effects: Background = ON, Color = White(#FFFFFF), Transparency = 0%
                Visual Boarder = ON, Color = Grey(#E6E6E6)
Tooltips = ON, Type = Report Page, Page = Auto

·        Add a legend to the map.
                                                
 In the report view, Select Insert ribbon and click on Image, load it from your device and resize a little bit and select, scaling to ‘FIT’
  In the Formatting Tab click on General, Properties, Size: Height =33, Width = 155, Position: Horizontal = 776, Vertical = 653

  To create the Map Legend select Insert ribbon, click on Text box Type ‘Low and High’, Highlight the Text and adjust Size = 14, Font = Segoe (Bold),  color=  Black
 In the Formatting Tab, General, Properties, Size: Height=33, Width=196, Position, Horizontal= 776, Vertical=686. Effects= Transparency= 100%
·       Move the legend to the bottom of the map.
